# 🌐 NeuroCards - Frontend

Modern UI for NeuroCards built with **React + Vite + TailwindCSS**.

---

## 🚀 Features

* 📄 Upload PDF
* 📚 Deck Management
* 🔄 Flashcard Flip UI
* 🎯 Review System (Again / Good / Easy)
* 📊 Progress Dashboard
* 🌈 Light professional UI

---

## 🛠️ Tech Stack

* React (TypeScript)
* Vite
* Tailwind CSS
* shadcn/ui
* Axios / Fetch API

---

## 📂 Project Structure

```
src/
 ├── components/
 ├── pages/
 ├── lib/api.ts
 ├── hooks/
```

---

## ⚙️ Setup Instructions

### 1. Install dependencies

```
npm install
```

---

### 2. Run project

```
npm run dev
```

Runs on:

```
http://localhost:8090
```

---

## 🔗 Backend Connection

Make sure backend is running:

```
http://localhost:8080
```

Update API base URL in:

```
src/lib/api.ts
```

Example:

```
const BASE_URL = "http://localhost:8080/api";
```
